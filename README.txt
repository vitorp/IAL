# Rotações e Reflexões no plano
## Simulação de Feixes de luz refletindo em uma parabola

## Configurações
### Ambiente
- python versão 3

### Dependências
Única Dependência é a biblioteca pygame que pode ser instalada com o comando "pip install pygame==2.6.1"

### Execução
Para executar o programa utilize o comando "python Reflexao_parabola.py"

## Controles

### Tela "Menu"
- Setas para cima e para baixo: Navegar menu
- Esc: fecha o programa

### Telas "Unico feixe" & "Varios feixes"
- Esc: retorna ao menu
- Setas para esquerda e direita: Altera angulo do feixe de luz

### Tela "Configurar origem"
- 1: Move origem para coordenada padrão
- 2: Move origem para o foco da parabola
- Setas: Move posição da origem
- Esc: retorna ao menu com posição da origem atualizada
